# Langchain-Beginners
Updated and modernized a 2-year-old LangChain tutorial project from Kaggle for compatibility with the latest LangChain version. Covers LLM Chains, Sequential Chains (Simple &amp; Complex), and Router Chains with cleaner structure and updated syntax. Original concept credits to Youssef19 on Kaggle.
Original Code Link : https://www.kaggle.com/code/youssef19/understanding-langchain-chains


Install the packages using the below command
uv add -r .\requirements.txt

Create the .env file and paste your Groq API key with the variable name mentioned below:

GROQ_API_KEY="Enter your API Key Here"

run the main.py file code.
